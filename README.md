## ⚡ Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Vanilla CSS & Modern Graphics
- **Focus:** Performance, Accessibility & Aesthetic

## 🛠️ Quick Start
```bash
npm install
npm run dev
```
Site will be live at `http://localhost:3000/Koinbasket`.

## 📦 Static Export
```bash
npm run build
```
Static assets generated in `/out`.
